== Installation ==
This section describes how to install the HT Newsletter for Elementor Page Builder Plugins for WordPress get it working.

= 1) Install =

= Install: =
1. Go to the WordPress Dashboard "Add New Plugin" section.
2. Search For "HT Newsletter for Elementor".
3. Install, then Activate it.

= OR: =
1. Unzip (if it is zipped) and Upload `ht-newsletter-for-elementor` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
